<p id="description">Clinic Management Software is a lightweight system designed to digitize basic operations of a dental clinic. It enables patient registration record management and appointment scheduling through a simple web interface built with HTML CSS and JavaScript. The backend is implemented in C where patient and appointment data are processed and stored in CSV files. The project focuses on providing a cost-effective simple and offline-capable solution for small clinics that need digital record management without complex ERP systems.</p>

<h2>Features</h2>

*   Patient Registration: Add and store new patient details in a structured CSV file.
*   Patient Record Management: Update and maintain patient information efficiently.
*   Appointment Scheduling: Book and manage appointments through a simple interface.
*   Calendar-Based Appointment View: Display scheduled appointments in an organized calendar layout.
*   CSV-Based Data Storage: Lightweight file-based storage without requiring a database.
*   C-Based Backend Logic: Handles patient and appointment data processing.
*   Web-Based User Interface: Clean frontend built with HTML CSS and JavaScript for easy interaction.
*   Simple and Cost-Effective Design: Suitable for small clinics needing basic digital record management.
